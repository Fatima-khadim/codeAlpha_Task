# Hangman Game (Python)

## Description
A simple text-based Hangman game where the player guesses a word one letter at a time.
The player can make up to 6 incorrect guesses.

## Features
- 5 predefined words
- Maximum 6 wrong attempts
- Shows guessed letters
- Random word selection

## Concepts Used
- random module
- while loop
- if-else statements
- strings and lists

## How to Run
Run the file using:

python hangman.py
