# CodeAlpha Chatbot (Python)

## Description
A simple interactive Python chatbot that responds to user input with predefined replies.

## Inputs & Replies
- `hello` → Hello  
- `how are you` → I am fine, thanks  
- `bye` → Goodbye  

## Features
- Uses functions, loops, and if-elif statements  
- Handles user input/output  
- Structured for easy readability and extension  

## How to Run
```bash
cd CodeAlpha_Chatbot
python chatbot.py
