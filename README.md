# CodeAlpha Tasks

This repository contains the internship projects for **CodeAlpha**.  

## Projects / Folders

- **CodeAlpha_Chatbot** — Python chatbot with predefined replies  
- **Hangman_Game** — Text-based Hangman game  
- **Stock_Portfolio_Tracker** — Tracks user's stock investments using predefined prices  

## How to Run
1. Navigate to the folder of the project you want to run:
```bash
cd CodeAlpha_Chatbot          # or Hangman_Game, Stock_Portfolio_Tracker
python chatbot.py              # or hangman.py, stock_tracker.py
