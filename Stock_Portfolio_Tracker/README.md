# Stock Portfolio Tracker (Python)

## Description
A simple Python program that calculates a user's total stock investment based on predefined stock prices.

## Features
- User inputs stock symbols and quantity  
- Hardcoded stock prices dictionary (e.g., {"AAPL": 180, "TSLA": 250})  
- Calculates total investment automatically  
- Shows each stock's total value  

## Concepts Used
- Dictionaries  
- Input/Output  
- Loops  
- Basic arithmetic  

## How to Run
```bash
cd Stock_Portfolio_Tracker
python stock_tracker.py
